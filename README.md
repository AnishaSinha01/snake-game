# Snake Game 🐍

A classic Snake Game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Smooth snake movement
- Food generation logic
- Collision detection (wall & self)
- Score tracking
- High score using localStorage
- Sound effects (eat, move, game over)
- Responsive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open index.html in your browser
3. Use arrow keys to play

## 🎮 Controls
- ⬆️ Arrow Up
- ⬇️ Arrow Down
- ⬅️ Arrow Left
- ➡️ Arrow Right
